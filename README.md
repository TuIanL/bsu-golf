# 高尔夫挥杆实时识别 MVP（Web）

## 如何运行
由于浏览器需要安全上下文才能使用摄像头，请用本地静态服务启动（不要直接 `file://` 打开）。

```bash
cd "/Users/hhl/Desktop/高尔夫姿态"
python3 -m http.server 8000
```

打开：
- `http://localhost:8000/web-app/index.html`

## 主要功能
- 一键选择：球杆（木/铁/挖起）、视角（正面/侧面）、持杆手（左右手）
- 摄像头实时姿态骨架叠加（MediaPipe Pose）
- 基于关键帧的挥杆分段与 Top10 问题诊断（启发式规则）
- 100 分制评分与扣分项列表
- 图文矫正建议（当前覆盖与 MVP 诊断一致的高频问题）
- 轨迹与重心 2D 代理可视化

## 备注（MVP限制）
- 由于只依赖单摄像头 2D 姿态，本版本的“打厚/打顶/曲球方向”等均为启发式代理指标，准确度会随光照、遮挡和站位变化。
- 下一阶段建议：补齐 30+ 问题标签（规则/模型）、引入训练数据并把关键参数从 2D 代理升级到更稳定的几何特征/3D估计。
